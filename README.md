# Test Task For Genesis
## Install
```shell
npm install
```
## Usage
### Run migrations
```shell
knex migrate:latest
```
### Run seeds(fixtures)
```shell
knex seed:run
```
### Task 1
```shell
node task1
```
### Task 2
```shell
npm run start
```
